Examples

load from a control file

>>> import gsflow
>>> control = gsflow.ControlFile.load_from_file("gsflow.gsflow")
>>> stats = StatVar.load_from_control_object(control)

load from a statistics file

>>> stats = StatVar("mystatvar.txt")
